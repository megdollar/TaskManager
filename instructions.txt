## Instructions to Run the Project

1. Ensure you have PHP installed on your machine. You can download it from https://www.php.net/downloads
2. Ensure you have XAMPP installed on your machine. You can download it here https://www.apachefriends.org/download.html
3. Extract the folder and place in your htdocs folder within your XAMPP application
4. Run your servers on XAMPP 
5. In XAMPP run the SQL queries in create_tables.sql to create a new database 'taskminder' and the associated create_tables
6. Verify the database and tables are correct in your local phpmyadmin http://localhost/phpmyadmin/index.php?route=/database/structure&db=taskminder
7. Open the index.php page in your localhost http://localhost/taskminder/views/index.php
8. Create an account, log in, edit account, log out are all working as part of the User class
